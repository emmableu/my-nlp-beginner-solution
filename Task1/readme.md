## 任务一：基于机器学习的文本分类

**实验目的**

实现基于logistic/softmax regression的文本分类

**实验内容：**
1）简单文本特征表示（词频、n-gram、TF-IDF）、特征组合与特征选择
2）模型相关：（随机）梯度下降、损失函数、学习率
3）实际操作：训练集/验证集/测试集的划分、shuffle

**补充：**
1）更加复杂的句子级特征暂时没有构建
2）可以使用纯numpy实现，比较懒所以用现成的库了
